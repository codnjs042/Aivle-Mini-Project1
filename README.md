# Aivle-Mini-Project1

### 일상대화 30 + 에이블스쿨 23
- 총 53가지의 답변이 가능한 챗봇  

### 초기 모델
- 텍스트 증강(koEDA), 토큰화(Mecab), 단어 임베딩(Word2Vec)을 활용하여 모델 구축
- Random Forest와 LGBM을 사용하여 성능 평가

### 개선 모델
- 맞춤법 검사(py-hanspell), 텍스트 증강(K-TACC), 토큰화(Okt), 어간 추출, 정규화, 불용어 제거 등 추가적인 전처리 작업을 거쳐 성능 개선
- 최대 정확도 73%
